# CONTRIBUTING & Safe Usage

Thanks for looking at this portfolio project. If you'd like to contribute or adapt these materials for training, please follow these rules:

- Use this repository for educational and demonstration purposes only.
- Do not include real credentials, passwords, or private data in commits.
- Test any detection logic in a sandbox or internal test environment before applying to production.
- If you submit changes that include IOCs, document their provenance and ensure they are sanitized.

To propose a change: open a pull request with a short description of your edits and include SAFETY.md compliance confirmation.
